HTML

HTML (HyperText Markup Language en ingles) es el lenguaje de marcado estandar para crear paginas web en cualquier editor de texto, es su estructura. Consta de una serie de elementos(HTML) que le dicen al navegador como mostrar el contenido por medio de las etiquetas.

Elemetnos HTML

Un elemento HTMl es el conjunto conformado por la etiqueta de inicio <etiqueta>, el contenido (h1, body, p, etc) y la etiqueta de cierre </etiqueta>.El nombre de las etiqeutas puede ir tanto en <minuscula> como en <MAYUSCULA>

A continuacion se mostrara los elementos HTML mas comunes.

<!DOCTYPE html>

Este elementos define que este archivo es un documento HTML.

<html>

Es el elemento base de una pagina HTML.

<head>

Es el elemento que contiene la meta informacion de la pagina.

<title>

Este elemento define el titulo de la pagina web que se vera en la pestaña del navegador.

<body>

Es el elemento que define el cuerpo del documento y que muestra todos los elementos visibles(fotos,videos,tablas, links, etc).

<h1, h2, h3, h4, h5, h6>

Es el elemento que define los encabezados y van desde el 1 (el pirnicpal) hasta el 6(menos important).

<p>

Es el elemento que define los parrafos.

<a>

Es el elemento que define los links o enlaces hacia otras paginas.

<img>

Es el elemento que define las imagenes, es uan etiqueta vacia.

Los elementos HTML pueden contener otro elementos, por ejemplo dentro de elemento <body> puede haber otro elementos como el <h1>, <p>, <section>, etc.

No olvidar poner la </etiqueta de cierre> a menos que sea un elemento vacio como <img>.

Formato de texto HTML

Las siguientes elementos HTML sirven para darle importancia a un texto, ya sea resaltandolo, subrayando, cambiando el tañama o en negrita.

El elemento HTML <b> muestra el texto en negrita (bold text en ingles) sin importancia extra.

El elemento HTML <strong> muestra el texto generlamente en negrita (bold text en ingles) y lo remarca como importante.

El elemento HTML <i> muestra el texto en cursiva (italic). Se suele usar para indicar un termino tecnico, una palabra o frase en otro idioma, un pensamiento, el nombre de una embarcacion, etc.

El elemento HTML <em> muestra el texto enfatizado(emphasized) y se muestra en cursiva (italic). Se suele usar para indicar lectura con otro enfasis.

El elemento HTML <small> muestra el texto pequeño.

El elemento HTML <mark> resalta el texto.

El elemento HTML <del> define el texto que fue eliminado (delete) de un documento. Los navegadores lo muentran tachando el texto con una linea.

El elemento HTML <ins> define el texto que fue insertado en un documento. Los navegadores lo muentran subrayando el texto.

El elemento HTML <sub> define el texto en subindice o subindicado. Este tipo de texto aparece medio caracter por debajo de la linea normal y mas pequeña, se usa para escribir formulas quimicas H2O.

El elemento HTML <sup> define el texto en superindice o superindicado. Este tipo de texto aparece medio caracter por arriba de la linea normal y mas pequeña, se usa para escribir notas al pie. Ikigai1

CSS

Si el HTML da estructura a la pagina web, elCSS(hojas de estilo en cascada por sus siglas en ingles) le da estilo al sitio cambiandole los colores del fondo y del texto. Tambien permite modificar la fuente y el tamaño de esta asi como la distrucion y el espacio entre elementos

IMPORTANTE: Se dice que el estilo se aplica en cascada porque si se aplica un background-color: red; a un <section> o <div>(elemento padre) se aplicara lo mismo a todos los elementos dentro de ese elemento (hijos)

Usando CSS

Hay 3 formas de usar el CSS

  • Online o en linea: Se usa para aplicar un estilo a un solo elemento HTML a traves del atributo style.
    Por ejemplo: <body background-color:red;></body>

  • Interno: se usa para aplicar estilo a una sola pagina web, se escribe el style en el <head>.
    Ejemplo: 
    <head>
<style>
body {background-color: powderblue;}
h1   {color: blue;}
p    {color: red;}
<style>
</head>
    En este caso se va a aplicar un color de fondo azul y el h1 y el p va a tener un texto de color azul y rojo respectivamente

  • Externo: Sirve para darle estilo a varias paginas, para usar una hoja de estilo externa se debe agregar un link de cada pagina en el <head>.

    Ejemplo: 
    <head>
<link rel="stylesheet" href="styles.css">
</head>

    En este caso la la hoja de estilo se vincula en la misma carpeta que la pagina actual. Tambien peude ser enlazada con una unicada en ka carpeta HTML del sitio web local (<link rel="stylesheet" href="/html/styles.css">) o por una URL (<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">)

    La hoja de CSS puede escribirse en cualquier editor de texto, no debe contenr ningun codigo HTML y debe ser guardado como un archivo .css

    Asi se ve un y escribe en un archivo style.css

Mapa de imagenes HTML

Con la etiqeuta map se peude crear un mapa de imagenes, significa que es una imagen con areas donde puede el usuario puede hacer click y segun donde se lo haga, hara difetenres cosas. A continuacion haremos un mapa de imagenes

1ro ponemos la imagen usando el elemento <img> y ademas el atributo usemap asi: <img src="workplace.jpg" alt="Workplace" usemap="#workmap">. El valor usemap inicia con # y el nombre del mapa de imagenes para crear el link entre la iamgen y el mapa.

Luego agregamos el elemento <map>. Este elemento sirve para crear el mapa y se vincula a la imagen con el atributo name que debe tener el mismo nombre que el usemap. Ejemplo <map name="#wormap">

Ahora debemos agregar las areas donde poder hacer click con el elemento <area>

Formas (shapes) de las areas

Las areas donde se debe dar click se debe definir con el atributo shape y uno de estos valores:

  • rect: Define un area rectangular. Se toma las coordenadas (<coords>) de la esquina superior izquierda y la inferior derecha y se anota los pares por ejemplo 22,30,170,150.
  • circle: Define un area circular. Se debe tomar la coordenada del centro del circulo y luego el radio del circulo. Se anota asi 445, 270,44
  • poly: Define una area poligonal. De esta manera se puee crear cualquier figura a traves de lineas rectas (poligonos)
  • default: Defina toda la region
Notas: Para cosneguir las coordenadas de una iamgen se debe usar algun programa de diseño grafico (GIMP, photoshop,etc). los numeros marcados en rojo representan las coordendas del eje x y se toman desde el margen izquierdo hasta el punto y los numeros rojos representan el eje Y, se toma dede el borde superior.
