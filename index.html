HTML

HTML (HyperText Markup Language en ingles) es el lenguaje de marcado estandar para crear paginas web, es su estructura. Consta de una serie de elementos(HTML) que le dicen al navegador como mostrar el contenido por medio de las etiquetas.

Un elemento HTMl e sel conjunto de etiqueta de inicio(<), contenido (h1, body, p, etc) y etiqueta de cierre (>). A continuacion se mostrara los elementos HTML mas comunes.


<!DOCTYPE html>

Este elementos define que este archivo es un documento HTML

<title>


<body>
